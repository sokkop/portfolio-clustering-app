## Portfolio Clustering App

Приложение для формирования диверсифицированного портфеля акций с использованием методов кластерного анализа.



Проект реализует подход к построению портфеля на основе кластеризации активов. Идея заключается в том, чтобы сгруппировать акции по схожести поведения и выбрать наиболее эффективные инструменты внутри каждого кластера.

Такой подход позволяет снизить внутрипортфельную корреляцию и повысить уровень диверсификации по сравнению с наивными стратегиями.


## Функциональность

* Загрузка CSV-файла с историческими ценами акций
* Предобработка данных:

  * обработка пропусков
  * корректировка сплитов и деноминаций
* Расчёт логарифмических доходностей
* Применение методов кластеризации:

  * K-Means
  * PCA + K-Means
  * Иерархическая кластеризация (single, complete, average)
* Формирование портфеля:

  * выбор акций с максимальным Sharpe Ratio в каждом кластере
  * расчёт весов по методу Risk Parity
* Сравнение результатов:

  * со случайным портфелем
  * с равновзвешенным рыночным портфелем
* Визуализация годовой волатильности портфелей

## Интерфейс

Приложение реализовано с графическим интерфейсом на tkinter.
Пользователь может:

* выбрать файл с данными
* задать дату разделения выборки
* выбрать метод кластеризации
* просмотреть результаты расчётов и сравнений


## Формат входных данных

CSV-файл должен иметь следующую структуру:

* индекс — даты (формат YYYY-MM-DD)
* столбцы — тикеры акций
* значения — цены закрытия

Пример:

```
Date,SBER,GAZP,LKOH
2023-01-01,100,200,300
2023-01-02,101,198,305
```
